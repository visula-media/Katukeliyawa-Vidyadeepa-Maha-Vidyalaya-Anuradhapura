# Katukeliyawa-Vidyadeepa-Maha-Vidyalaya-Anuradhapura
Katukeliyawa Vidyadeepa Maha Vidyalaya in Anuradhapura provides quality education from Grade 1 to Grade 11. We nurture academic excellence, discipline, and strong values in a supportive environment, helping students grow into confident, responsible, and well-rounded individuals prepared for the future.
